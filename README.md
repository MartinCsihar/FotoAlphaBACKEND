# Short description
This repository contains the backend of my full‑stack application built with the Spring Boot framework.

# My project: FotoAlpha
As the name suggests it the topic is photography. On the website clients can register a profile and easealy log in.
A logged in user can view it's photos or videos that was made for them, and naturally users can also view their appointments because a logged in user can create an appointment.

## Technologies used:
  - Spring Boot
  - Spring Security
  - Eureka
  - Cloud Gateway
  - OAuth2
  - Kafka
  - Docker
  - Java Mail Sender
  - AWS S3
  - PostgreSQL
  - GIT
  - Microservices
